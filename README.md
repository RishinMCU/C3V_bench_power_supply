# C3V_bench_power_supply
Its a simple Linear bench power supply (0-12)VDC (0-1.6)ADC 
Requirements: 

1. 0-12v and 0-1.6amp dc out
2. Analog control with digital voltage and current display
3. made from "jelly beans" parts
4. Linear
